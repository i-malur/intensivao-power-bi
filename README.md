# Intensivão de Power BI
Entre os dias 15/12/2025 a 18/12/2025, tive 4 aulas criando 4 dashboards diferentes no PowerBI

## Aula 01 - Dashboard de Faturamento
* Instalando o Power BI
* Importando e tratando dados
* Criando primeiros visuais usando cartões e gráficos de barras
* Utilizando o gráfico de mapas

Neste relatório podemos observar:
* Faturamento e quantidade vendida por ano e mês
* Faturamento total
* Produto mais vendido
* Desempenho de vendas por marca
* Desempenho de faturamento por continente


## Aula 02 - Dashboard de Produção
* Alterando o tipo dos dados
* Removendo linhas vazias
* Realizando operações matemáticas e criando novas colunas
* Transformando dados
* Dividindo colunas
* Criando medidas
* Adicionando plano de fundo e estilizando o relatório
* Usando os sites: [My Color Space](https://mycolor.space/) e [Color HEX](https://www.color-hex.com/)
* Criando segmento de dados
* Estilizando o relatório com ícones

Neste relatório podemos observar:
* Horas produtivas
* Horas paradas
* Quantidade de produtos aprovados
* Quantidade de produtos rejeitados
* Disponibilidade => porcentagem de horário trabalhado (horas produzidas / horas trabalhadas)
* % Qualidade => porcentagem de produtos aprovados
* Total de produtos aprovados por mês
* Segmentação com filtro de funcionários
* Segmentação com filtro de meses

## Aula 03 - Dashboard de RH
* Separando dados em colunas
* Criando medidas
* Estilizando relatório com plano de fundo
* Utilizando Sparkline para criar gráficos com a porcentagem de contratações
* Utilizando Árvore Hierárquica
* Criando Tooltip

Neste relatório podemos observar:
* % de contratações
* Funcionários: ativos, desligados e % rotatividade
* Funcionários ativos e sálario por cidade
* Árvore Hierárquica de funcionários ativos por setor e cargo
* Tooltip por cidade com informações de funcionários ativos, somatório de salários e porcentagem de horas extras por cargo 

## Aula 04 - Dashboard Financeiro
* Criando medidas
* Estilizando relatório
* Usando o Scroller, Image Grid, Enlighten Data Story

Neste relatório podemos observar:
* Recebimentos => Valores de movimentações de recebimento
* Custos => Valores de movimentações de pagamento
* Pagamento de impostos
* Lucro total
* Margem de lucro
* Lucro mensal
* Imagens das logos dos bancos para a criação de filtro por banco
* Letreiro com margem de lucro por municipio
* Letreiro com movimentação via PIX
